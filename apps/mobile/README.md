# Sonic Music Mobile (uni-app x)

Mobile application built with uni-app x.

## Coming Soon

This mobile app will be developed using uni-app x framework.

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev:mp-weixin  # WeChat Mini Program
npm run dev:h5         # H5
npm run dev:app        # App

# Build
npm run build:mp-weixin
npm run build:h5
npm run build:app
```

## Tech Stack

- uni-app x
- Vue 3
- TypeScript
- Shared packages from monorepo
